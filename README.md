# Cloud ETL Automation Pipeline

Automated ETL pipeline using Python, Apache Airflow, AWS S3, PySpark, Docker and PostgreSQL.

## Features

- Automated data extraction
- Data cleaning and transformation
- PostgreSQL loading
- Apache Airflow orchestration
- AWS S3 integration
- PySpark processing
- Dockerized deployment

## Tech Stack

- Python
- Pandas
- PySpark
- Airflow
- PostgreSQL
- AWS S3
- Docker

## Project Flow

CSV Dataset
→ Extract
→ Transform
→ Load
→ PostgreSQL

## Future Improvements

- AWS Redshift warehouse loading
- CI/CD using GitHub Actions
- Monitoring with Grafana
